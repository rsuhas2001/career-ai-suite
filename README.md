# Multi-Agent LLM Career AI Suite

A multi-agent AI platform built with GPT-3.5-Turbo and Gradio that automates resume scoring and mock interview feedback through a system of specialized agents coordinated by a Python orchestrator.

## Features
- 6 specialized AI agents handling distinct tasks
- Resume scoring with role-specific criteria
- Mock interview generation and live feedback
- Response times of 2–5 seconds
- Gradio interface for interactive use

## Tech Stack
Python, OpenAI GPT-3.5-Turbo, Gradio, Custom Python Orchestrator

## Setup
git clone https://github.com/rsuhas2001/career-ai-suite.git
cd career-ai-suite
pip install -r requirements.txt
export OPENAI_API_KEY=your_key_here
python main.py
